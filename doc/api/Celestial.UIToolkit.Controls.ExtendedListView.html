Class ExtendedListView

An extension of the ListView control which provides events and methods dealing with its Item-Containers.

Remarks

This custom list view is mainly intended to be used by the NavigationView. Since the functionality here is universal and not specific to the NavigationView, it is provided as a publicly available class.

Note that it is not designed to work performantly with a large amount of items. The normal ListView does a better job at that.
