Class ExtendedVisualStateManager

A custom visual state manager which provides the same features, aswell as additional ones, as the default VisualStateManager.

Inheritance

Namespace: Celestial.UIToolkit.Xaml

Syntax
public class ExtendedVisualStateManager : VisualStateManager
Remarks

For using this extended visual state manager, the CustomVisualStateManagerProperty dependency property must be set on a control which defines the visual state groups (-> the state groups root element).
