Class SharedResourceDictionary

An extension of the ResourceDictionary class which shadows the Source property. The difference to a normal resource dictionary is that when this class' Source property is set, the dictionary from that source gets loaded via the Celestial.UIToolkit.Xaml.SharedResourceDictionaryManager, which allows caching of the resource dictionaries. The loaded resource dictionary set by the Source property is then accessible via the MergedDictionaries property.
