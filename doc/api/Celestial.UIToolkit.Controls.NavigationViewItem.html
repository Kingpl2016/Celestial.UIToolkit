Class NavigationViewItem

Represents the container for a generic navigation item in a NavigationView.

Syntax
public class NavigationViewItem : ListViewItem, DUCE.IResource, IAnimatable, IFrameworkInputElement, IInputElement, ISupportInitialize, IHaveResources, IQueryAmbient, IAddChild
Remarks

This class is inheriting from ListViewItem since the NavigationView is using a ListView to display its menu items. To ensure that a custom template will still display these items correctly, it should use a ListView aswell.
